React Product Catalog
Описание проекта

Это простой каталог товаров, реализованный на React с использованием mock-API.

Установка

Клонируйте репозиторий:

git clone <repo_url>


Перейдите в папку проекта:

cd react-product-catalog


Установите зависимости:

npm install


Запустите проект:

npm start


Приложение откроется по адресу http://localhost:3000.

Использование

Вводите название товара в поле поиска — список фильтруется в реальном времени.

Нажмите на карточку товара — откроется модальное окно с подробной информацией.

Закройте модальное окно кнопкой × или кликом по затемнённому фону.
